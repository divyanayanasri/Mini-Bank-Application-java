# Mini Bank Application (Java - OOP Based)

## 📌 Project Description
This project is a **command-line based Mini Bank Application** developed using **Core Java and Object-Oriented Programming (OOP)** concepts.
It simulates basic banking operations where a user can create an account and perform transactions through a menu-driven interface.
---
## 🚀 Features
-  Create Account (Name & Bank ID)
-  Deposit Money
-  Withdraw Money
-  Check Account Balance
-  View Previous Transaction
-  Calculate Interest Over Time
-  Exit Option
---
##  Concepts Used
- Object-Oriented Programming (OOP)
- Encapsulation
- Reusability
- Classes & Objects
- Methods
- Conditional Statements
- Loops (Menu-driven system)
- Java Input Handling (Scanner)
---
## ⚙️ Technologies Used
- Java
- VS Code / Eclipse
- JDK
---
## 📂 Project Structure
MiniBankApp/|---Account.java|---Bank.java
---
##  How to Run
1. Compile the files:
     javac Account.java Bank.java
2. Run the program:
     java Bank
---
##  Output / Flow
- User enters Name & ID
- Menu is displayed:
  - Check Balance
  - Deposit
  - Withdraw
  - Previous Transaction
  - Interest
  - Exit
- Operations repeat until user exits
